Let the visitors post stories, articles and news direct to your site. The basic functions to deliver Social Journalism experience.

== Description ==

Version 1.1 includes lots of bug fixing and removal of the theme, now any theme can use the social-news plugin, enjoy!

This plugin adds a new post type and a form so that your site visitors can post stories, news or articles direct to the site (no, they wont have to be registered). Live usage at http://www.poetleaks.com/add/

**The work flow**
The "Social News" plugin have an work flow where the posts will go in to WordPress in draft mode, and in the "Social News" post type. The blog editor will be notified by email, he/she reviews the post, add tags, categorys and in the end publish or removes it. The post author will be notified that the post is polished and receive an email containing a link back to the post.

**Options**
The post form can use reCaptch and Facebook for security and to relate a person and user picture to the post. If so you will need to add some "keys" as template options.

== Installation ==

1. Drop plugin in the plugin folder and activate
3. Now you have a domain.com/add path to add news
2. Add optional template settings, Facebook API-key, reCaptcha key
3. Done
